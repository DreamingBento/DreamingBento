--- layout: post type: update title: Item System permalink: playto/updates/itemsystem/ date: 2017-11-02 author: Neil email: neilnthings@gmail.com summary: Hey everyone!

I was able to create a basic item system. The code is kept in the game controller script which is shared through all the scenes. The items are referenced in each scene and then new abilities can be used depending on which items the player currently has on the player inventory.

Players can only carry three items at a time and any other items they may get if their inventory is full will go to there inventory box at the home base. prev: playto/updates/watermechanics/ next: playto/updates/healthsystem/ ---
